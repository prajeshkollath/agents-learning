# AI Vault

This is your Obsidian vault for AI notes, living at `c:\Users\freed\.vscode\Agents\AI`.

## Quick start

- **Open quick switcher:** `Ctrl+O`
- **Global search:** `Ctrl+Shift+F`
- **Command palette:** `Ctrl+P`
- **Open graph view:** click the graph icon in the left ribbon
- **Create a link:** type `[[` and start typing a note name

## How it works

Every note is a plain Markdown file in this folder. Linking with `[[note name]]`
builds the graph and backlinks automatically. YAML frontmatter (the block at the
top of this file) becomes queryable **properties**.

## Suggested structure

- `Topics/` — one note per concept
- `Projects/` — project-specific notes
- `Daily/` — daily notes (the Daily Notes core plugin is enabled)
- `attachments/` — images/PDFs you paste in

Delete this note whenever you like.
